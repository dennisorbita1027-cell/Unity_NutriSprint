# Unity_NutriSprint

A 3D mobile endless runner game developed as part of my undergraduate thesis project at Nueva Vizcaya State University. The project explores how interactive gaming mechanics can improve children’s awareness of proper nutrition and healthy food choices.



## Core Algorithms & Architecture


#### Procedural Content Generation (PCG)
To provide an endless gameplay experience without manual scene designing, the game dynamically spawns track modules, obstacles, and nutritional items. 

#### Dynamic Difficulty Adjustment (DDA)
The gameplay automatically scales its difficulty dynamically based on a **time-driven algorithm** to keep players in a state of optimal challenge engagement.

#### Performance Tracking System
A robust backend data tracker that records user performance in real-time which serves as the primary data collection layer to validate the study's educational impact.

#### Achievement System
An event-driven progression tracker designed to sustain pupils' interest and significantly enhance the game's replayability.


## Tech Stack & Specifications
*   **Engine:** Unity 
*   **Language:** C#
